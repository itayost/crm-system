🚀 הטמעת טופס לידים ב-CRM

3 דרכים להטמיע את הטופס באתר שלך

הוראות הטמעה

שיטה 1: הטמעה באמצעות iframe (הכי פשוט)

העתק והדבק את הקוד הבא בכל מקום באתר שלך:

<iframe src="https://crm-system-alpha-eight.vercel.app/lead-form.html" width="100%" height="800" frameborder="0" style="max-width: 500px;" ></iframe>
יתרונות: התקנה פשוטה, עובד בכל אתר, מתעדכן אוטומטית

שיטה 2: הטמעת HTML ישירות

הוסף את הקישורים הבאים ב-<head> של האתר:

<link href="https://fonts.googleapis.com/css2?family=Heebo:wght@300;400;500;600;700&display=swap" rel="stylesheet">

והעתק את ה-HTML של הטופס למקום הרצוי:

<div class="crm-lead-form"> <!-- הורד את הקובץ lead-form.html וצפה בקוד המלא --> </div> <script src="https://crm-system-alpha-eight.vercel.app/lead-form.js"></script>
יתרונות: שליטה מלאה על העיצוב, אפשרות להתאמה אישית

שיטה 3: React Component

אם האתר שלך בנוי ב-React/Next.js, תוכל להשתמש ברכיב המוכן:

import LeadCaptureForm from '@/components/public/LeadCaptureForm' export default function LandingPage() { return ( <LeadCaptureForm apiUrl="https://crm-system-alpha-eight.vercel.app/api/public/leads" onSuccess={() => console.log('Lead submitted!')} /> ) }
יתרונות: אינטגרציה מלאה עם React, TypeScript support

⚙️ הגדרות נוספות

כדי לשנות את כתובת השליחה (אם אתה מריץ CRM משלך), ערוך את המשתנה:

const CRM_API_URL = 'https://your-crm-domain.com/api/public/leads';
✅ הטופס כולל:
  • ולידציה מובנית לכל השדות
  • תמיכה מלאה בעברית ו-RTL
  • הודעות הצלחה/שגיאה
  • CORS support לשליחה מכל דומיין
  • עיצוב רספונסיבי למובייל
